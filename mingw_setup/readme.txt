
Windows
=======

Note: For better readability, web links are at the bottom of this section

1. Download the latest MinGW installer from [1]

2. Install MinGW
    
    a. Select "Download latest repository catalogues" during installation.

    b. These instructions assume that you installed to C:\MinGW. If that's
       not the case, substitute where necessary.

    c. Select at least "MSYS Basic System" for installation. You can deselect
       all the rest if you want.
       

3.
